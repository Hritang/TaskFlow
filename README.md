# 📖 Overview

TaskFlow is a modern Kanban board application designed to simplify task management.

Users can create, edit, organize, and prioritize tasks using an intuitive drag-and-drop interface. Tasks are automatically saved using the browser's Local Storage, ensuring data persists across sessions without requiring a backend.

The project demonstrates modern React development practices, responsive UI design, reusable components, and efficient state management.

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Hritang/TaskFlow.git
```

### Navigate to the project

```bash
cd TaskFlow
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
